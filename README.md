# Spam Detection App

A Machine Learning based Spam Detection Web App built using Python, Scikit-learn, and Streamlit.

## Features
- Detects Spam and Not Spam messages
- Simple Streamlit UI
- Multinomial Naive Bayes model

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run Locally

```bash
streamlit run SpamDetection.py
